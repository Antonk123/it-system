## Layout Contract

### Backup Section Structure

The backup section follows the exact same Collapsible + Card pattern used by all existing Settings sections (Appearance, Users, Categories, Tags, etc.):

```
<Collapsible>
  <Card>
    <CollapsibleTrigger>
      <CardHeader className="cursor-pointer hover:bg-primary/10 transition-colors">
        <CardTitle>
          [Icon] Backup & Export
          <span className="ml-auto text-sm text-muted-foreground">+/−</span>
        </CardTitle>
        <CardDescription>
          Ladda ned en komplett kopia av databasen och uppladdade filer.
        </CardDescription>
      </CardHeader>
    </CollapsibleTrigger>
    <CollapsibleContent>
      <CardContent className="space-y-4">
        [description paragraph]
        [backup button]
        [last backup info if available]
      </CardContent>
    </CollapsibleContent>
  </Card>
</Collapsible>
```

### Backup Card Content Layout

Inside `CardContent`:

1. Description text (14px, muted-foreground): explains what the ZIP contains — database + uploaded files.
2. "Ladda ned backup" Button: variant `default`, size `default`, min-height 44px. Left icon: `HardDriveDownload`. Loading state: replace icon with `Loader2 animate-spin`.
3. File size display: after download initiates, a success toast shows the file size (per D-03).

## Interaction States

### Backup Button States

| State | Visual |
|-------|--------|
| Idle | `variant="default"` (primary color fill), label: "Ladda ned backup", icon: HardDriveDownload |
| Loading | Button disabled, icon replaced by `Loader2 className="animate-spin"`, label: "Genererar backup…" |
| Success | Button returns to idle; success toast fires via `toast.success()` with filename and file size |
| Error | Button returns to idle; error toast fires via `toast.error()` with failure message |

### Toast Copy

| Event | Toast type | Message |
|-------|-----------|---------|
| Download starts / blob ready | `toast.success` | "Backup skapad — {filename} ({size} MB)" |
| Network or server error | `toast.error` | "Backup misslyckades. Försök igen." |
