**Route 1: GET /dashboard-overview** (per D-08)

Add route handler `router.get('/dashboard-overview', authenticate, (req: AuthRequest, res: Response) => { ... })` with two SQL queries:

1. **Aging tickets query** (per D-03 — aging = days since last activity, NOT creation date):
```sql
SELECT
  t.id,
  t.title,
  t.priority,
  t.status,
  c.name as requester_name,
  CAST(julianday('now') - julianday(
    MAX(t.updated_at, COALESCE(
      (SELECT MAX(tc.created_at) FROM ticket_comments tc WHERE tc.ticket_id = t.id AND tc.deleted_at IS NULL),
      t.updated_at
    ))
  ) AS INTEGER) as age_days
FROM tickets t
LEFT JOIN contacts c ON t.requester_id = c.id
WHERE t.status IN ('open', 'in-progress', 'waiting')
ORDER BY age_days DESC
LIMIT 6
```
Use `LIMIT 6` so frontend can detect if "Visa alla" is needed (> 5 means show link). COALESCE handles tickets with no comments (avoids NULL in MAX — see Pitfall 2 in RESEARCH.md). Filter `tc.deleted_at IS NULL` to exclude soft-deleted comments (Pitfall 5).

2. **Today counts query** (per D-06):
```sql
SELECT
  SUM(CASE WHEN date(created_at) = date('now') THEN 1 ELSE 0 END) as created_today,
  SUM(CASE WHEN date(resolved_at) = date('now') THEN 1 ELSE 0 END) as resolved_today,
  SUM(CASE WHEN date(closed_at) = date('now') THEN 1 ELSE 0 END) as closed_today
FROM tickets
```

Return JSON: `{ agingTickets: AgingTicketRow[], todayCounts: { created_today: number, resolved_today: number, closed_today: number } }`

**File 1: src/hooks/useDashboardOverview.ts**

Create a new file following the exact pattern of `useReportsSummary.ts`. Contents:

```typescript
import { useQuery } from '@tanstack/react-query';
import { api } from '@/lib/api';

export interface AgingTicket {
  id: string;
  title: string;
  priority: string;
  status: string;
  requester_name: string | null;
  age_days: number;
}

export interface TodayCounts {
  created_today: number;
  resolved_today: number;
  closed_today: number;
}

export interface DashboardOverview {
  agingTickets: AgingTicket[];
  todayCounts: TodayCounts;
}

export const dashboardOverviewKeys = {
  all: ['dashboard', 'overview'] as const,
};

export const useDashboardOverview = () => {
  return useQuery<DashboardOverview>({
    queryKey: dashboardOverviewKeys.all,
    queryFn: () => api.request<DashboardOverview>('/tickets/dashboard-overview'),
    staleTime: 60 * 1000,
    gcTime: 5 * 60 * 1000,
  });
};
```

staleTime is 60 seconds (per UI-SPEC note 3) so dashboard data refreshes frequently. refetchOnWindowFocus defaults to true in React Query, handling stale midnight counts on long-open tabs (Pitfall 4 in RESEARCH.md).
