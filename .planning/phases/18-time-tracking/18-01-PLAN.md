    **2. src/lib/duration.ts — Create duration parser and formatter (per D-01, D-06)**

    Create new file with two exported functions:

    `parseDuration(input: string): number | null` — Parses human-readable duration to integer minutes. Trim and lowercase input first. Match in this order:
    - Decimal hours: `/^(\d+(?:\.\d+)?)\s*[ht]$/` — e.g. '1.5h', '2t' -> Math.round(float * 60)
    - Combined: `/^(\d+)\s*[ht]\s*(\d+)\s*m(?:in)?$/` — e.g. '1h 30m', '1t30m' -> hours*60+min
    - Minutes only: `/^(\d+)\s*m(?:in)?$/` — e.g. '90m', '45min' -> parseInt
    - Plain integer: `/^(\d+)$/` — e.g. '90' -> parseInt (assumed minutes)
    - Returns null if no match or result is 0

    `formatDuration(minutes: number): string` — Formats integer minutes:
    - minutes <= 0: return '0m'
    - minutes < 60: return `${minutes}m`
    - else: `const h = Math.floor(minutes / 60); const m = minutes % 60;` return m === 0 ? `${h}h` : `${h}h ${m}m`
